### *12. –ò—Ç–µ—Ä–∞—Ç–æ—Ä (Iterator)*

**–ò—Ç–µ—Ä–∞—Ç–æ—Ä** ‚Äî —ç—Ç–æ –æ–±—ä–µ–∫—Ç, –∫–æ—Ç–æ—Ä—ã–π –ø–æ–∑–≤–æ–ª—è–µ—Ç –ø–µ—Ä–µ–±–∏—Ä–∞—Ç—å —ç–ª–µ–º–µ–Ω—Ç—ã –∫–æ–ª–ª–µ–∫—Ü–∏–∏. –û–Ω –¥–æ–ª–∂–µ–Ω –∏–º–µ—Ç—å –º–µ—Ç–æ–¥—ã __iter__() –∏ __next__().

–ü—Ä–∏–º–µ—Ä:
```python

class MyIterator:
    def __init__(self, data):
        self.data = data
        self.index = 0

    def __iter__(self):
        return self

    def __next__(self):
        if self.index >= len(self.data):
            raise StopIteration
        value = self.data[self.index]
        self.index += 1
        return value

iterator = MyIterator([1, 2, 3])
for item in iterator:
    print(item)  # 1, 2, 3
```
